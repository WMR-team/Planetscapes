Coarse annotations

In addition to the fine annotations, we provide coarser ploygonal annotations for a set of 20,000 images in collaboration with XX. Again, overlayed colors encode the semantic classes (see class definations). Note that we do not aim to annotated single instances, however, we marked polygons covering individual objects as such.
